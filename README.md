# Finance Management System

A full-stack web application developed to manage personal finance activities such as tracking income, expenses, and financial records in an organized way.

## Features
- Add and manage income details
- Add and track expenses
- Categorize financial records
- View and manage transactions
- Backend APIs for handling finance data
- User-friendly frontend interface

## Tech Stack
### Frontend
- Next.js
- React
- TypeScript
- CSS / Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
